ملاحظة تنظيمية

تركيزي المهني الرئيسي هو الأمن السيبراني والشهادات المهنية المرتبطة به، وهذه الصفحة تعرض نماذج أعمال تطوير الويب كجزء عملي داعم لخبرتي التقنية.

كل عميل جديد يساهم في تطوير خبرتي وتحسين جودة مشاريعي، ويمكن تكييف نفس الأفكار لتناسب قطاعات مختلفة (طبي، تعليمي، مطاعم، خدمات، شركات).

المشاريع (مواقع ويب)

1) موقع شخصي / بورتفوليو

موقع احترافي مناسب للأفراد الراغبين في عرض سيرتهم الذاتية أو أعمالهم بشكل إلكتروني، مثل الطلاب والمهندسين والمستقلين.

شخصي متجاوب GitHub Pages
فتح الموقع

2) موقع مركز علاج طبيعي (SCC)

موقع مخصص لمراكز العلاج الطبيعي والمراكز الطبية، بتصميم واضح وسهل الاستخدام، مع نظام حجز مواعيد يرسل الطلبات مباشرة إلى البريد الإلكتروني.

طبي حجز مواعيد إرسال بريد
فتح الموقع

3) موقع شركة استشارات هندسية (R E.C Safety)

موقع لشركة متخصصة في الاستشارات بمجال السلامة والصحة المهنية، يتضمن تعريفاً بالخدمات، عرض الأهداف، باقات بأسعار واضحة، ونموذج تواصل/حجز سريع.

هندسي سلامة مهنية باقات وخدمات
فتح الموقع

4) موقع خدمات (Shlool Web)

منصة تتيح طلب تصميم موقع إلكتروني بخطوات بسيطة وسهلة، مع إمكانية التواصل المباشر وطلب الخدمات عبر الموقع.

خدمات طلب مباشر واجهة بسيطة
فتح الموقع رابط الطلب

5) منصة تعليمية (كنز العلم)

منصة تعليمية إلكترونية تعرض الدروس والمحتوى الأكاديمي بطريقة منظّمة وسهلة الاستخدام، مناسبة للمدارس والمراكز التعليمية.

تعليمي تنظيم محتوى واجهة سهلة
فتح الموقع

6) موقع مطعم

تصميم بسيط وجذّاب لعرض قائمة الطعام والعروض اليومية، مناسب للمطاعم والمقاهي الصغيرة الراغبة في واجهة أنيقة وسهلة التصفح.

مطاعم عروض يومية واجهة عملية

أعمال أخرى (ثانوية)

هذه الأعمال موجودة لكنها ليست محور العرض الرئيسي حاليًا، ويمكن إبرازها عند الحاجة حسب نوع العميل.
